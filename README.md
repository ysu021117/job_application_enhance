# AI Career Skills

Reusable skills for resume targeting, application strategy, and voice-aligned career document rewriting.

## Included Skills

### `job-application-strategy`

Use this skill to:

- extract and normalize experience from multiple resume versions
- preserve quantified evidence from resumes, transcripts, portfolios, and supporting materials
- analyze a target role and company context
- recommend what to emphasize in a targeted resume
- generate a markdown resume blueprint before final drafting

### `career-voice-match`

Use this skill to:

- revise resumes and cover letters so they sound more natural and human
- align the final writing with the user's own professional voice
- preserve factual accuracy while improving tone
- preserve Word formatting and page constraints when a reference document is supplied

## Suggested Workflow

1. Use `job-application-strategy` to consolidate source materials and create a targeted resume blueprint.
2. Review and approve the blueprint.
3. Use `career-voice-match` to produce the final resume wording in the user's voice.
4. If needed, produce a cover letter after the resume is approved.
