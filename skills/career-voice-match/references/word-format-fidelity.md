# Word Format Fidelity

Use this reference when revising a resume or cover letter that will be delivered as a Word document and the user wants it to match a reference document's layout.

## Goal

Preserve the reference document's formatting system while updating the wording and tone.

## Required Checks

Before delivery, confirm alignment on:

- font family
- font sizing pattern
- line spacing
- paragraph spacing
- margin pattern
- heading and subheading treatment
- section label style
- bullet style where relevant
- overall page count

## One-Page Rule

If the reference document is one page, the final rewritten document must remain one page.

Do not let cleaner wording expand the document into a second page by default.

If compression is needed:

- shorten wording
- remove redundancy
- compress lower-priority content only if the user allows content reduction

## Heading And Hierarchy

Preserve how the reference handles headings, including:

- hierarchy
- capitalization
- spacing
- emphasis
- alignment

If the document contains custom heading styles or section labels, keep them consistent in the final Word output.

## Priority Rule

When balancing style and format:

1. preserve facts
2. preserve required tone alignment
3. preserve the reference layout and page-count constraint
4. compress language before changing document structure
