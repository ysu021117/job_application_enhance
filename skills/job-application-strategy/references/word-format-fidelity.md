# Word Format Fidelity

Use this reference whenever the final deliverable is a Word document and the user has provided a reference document whose formatting should be followed.

## Goal

Make the final `.docx` output match the reference document's formatting system, not just its general style.

## Required Fidelity Areas

Match the reference document on:

- font family
- font sizing pattern
- line spacing
- paragraph spacing
- margin pattern
- indentation pattern
- heading and subheading treatment
- section label style
- bullet style where relevant
- overall page count

## Heading Rules

Preserve the reference document's heading system, including:

- section ordering
- capitalization style
- spacing above and below headings
- weight, size, and alignment differences between headings and body text

Do not rewrite the content in a way that breaks the reference document's heading structure unless the user explicitly asks for a structural redesign.

## Page-Length Rule

If the reference document is one page, the final rewritten document must also be one page.

Treat this as a hard constraint, not a preference.

If needed:

- tighten wording
- compress lower-priority content
- reduce redundancy
- make sharper content-selection choices

Do not allow the final document to spill into an extra page just because the new language is more polished.

## Priority Rule

When formatting fidelity conflicts with content volume:

1. preserve factual accuracy
2. preserve the highest-priority content for the target role
3. preserve the reference formatting and page-count constraint
4. omit or compress lower-priority content

## If No Reference Document Exists

If no formatting reference is supplied, use a clean professional document structure and state that no exact formatting fidelity target was available.
